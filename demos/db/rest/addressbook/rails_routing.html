Contacts
HTTP Verb Path Used for
OPTIONS /contacts Display a JSON schema for contacts table.
GET /contacts Display a list of all contacts.
GET /contacts/:id Display a specific contact.
POST /contacts Create a new contact.
PUT /contacts/:id Update a specific contact.
DELETE /contacts/:id Destroy a specific contact.
Phones
HTTP Verb Path Used for
OPTIONS /contacts/phones Display a JSON schema for phones table.
GET /contacts/:contactid/phones Display a list of all phones.
GET /contacts/:contactid/phones/:id Display a specific phone.
POST /contacts/:contactid/phones Create a new phone.
PUT /contacts/:contactid/phones/:id Update a specific phone.
DELETE /contacts/:contactid/phones/:id Destroy a specific phone.
