Please see: Rails Routing from the Outside In.

HTTP Verb Path Action Used for
GET /person Index Display a list of all persons.
GET /person/new New Return an HTML form for creating a new person.
POST /person Create Create a new person.
GET /person/:id Show Display a specific person.
GET /person/:id/edit Edit Return an HTML form for editing a person.
POST /person/:id Update Update a specific person.
GET /person/:id/delete Delete Return an HTML form for deleting a person.
POST /person/:id/destroy Destroy Destroy a specific person.
